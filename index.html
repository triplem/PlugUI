PlugUI 2

PlugUI 2 is a system admin web interface for small ARM based devices running Arch Linux ARM. It is based on the PlugUI module of the ArchLinux ARM community. They have decided to use NodeJS instead of an older Python implementation (see branch django in their github-repository). Therfore the whole system is now programmed in Javascript.

We are trying to establish this PlugUI as the de-facto standard admin interface for the ArchLinux ARM plattform. There are a couple of alternatives, which are not really "applicable" for the ArchLinux plattform (see Tonido Apps, RaspControl, ...), and we are going to make a better application and fully support the alarm system.

It should run on other distros as well, but the package manager is right now geared to use pacman. The system is pretty much modularized, so that it should be possible to use other package managers as well, but this is not going to be developed by us (at least in the near future).

The current version is under high development, it is a rewrite of the older PlugUI code, using Node.js. The older code is still in this repo in the django branch.
The system is split into a server part, and a client part. The server part provides a REST API to all offered functionalities, therefor it is possible to connect other systems to this application as well. The client part is based on backbone.js as well as Backbone.Marionette.
Both parts of the appliation are highly modularized, so that is is fairly easy to provide additional functionality and/or exchange already existing functionality. This modularization is one of the main reasons, why we decided to re-organize the application structure.
